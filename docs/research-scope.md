# Research Scope

This repository documents only the public-facing conceptual scope of an ongoing research direction.

The public scope includes:

- research themes
- conceptual questions
- general problem framing
- activity timeline

The following are intentionally excluded:

- protocol logic
- implementation design
- private specifications
- system internals
- verification architecture
- non-public technical materials

The purpose is to preserve a public-facing research log without disclosing core design details.
